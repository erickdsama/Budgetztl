# Design System Documentation

## Theme Overview

This document outlines the core visual theme of our design system, reflecting a modern and user-friendly aesthetic.

## Colors

### Palette
Our color palette is carefully selected to provide a clear hierarchy and maintain brand consistency.

- **Primary Color:** `#5f78a4` - A calm, foundational blue, used for key interactive elements, calls to action, and primary branding.
- **Secondary Color:** `#6e778a` - A soft, supportive gray-blue, for less prominent UI elements, secondary actions, and complimentary backgrounds.
- **Tertiary Color:** `#7f709b` - A subtle, accentuating purple, providing visual interest for highlights, badges, or decorative elements.
- **Neutral Color:** `#76777c` - A balanced grey, serving as the base for backgrounds, surfaces, and non-chromatic components.

### Color Mode
The system currently operates in a `light` mode, optimized for readability and a bright user experience.

## Typography

Our typography system prioritizes clarity and legibility across all content types.

- **Headline Font:** `Inter` - Used for all headlines, ensuring a strong visual impact and clear hierarchy.
- **Body Font:** `Inter` - Applied to all body text, chosen for its excellent readability at various sizes.
- **Label Font:** `Inter` - Utilized for labels and smaller text elements, maintaining consistency with other textual components.

## Shape and Spacing

### Roundedness
UI elements feature a `moderate` level of roundedness (value `2`), striking a balance between modern softness and structural clarity.

### Spacing
The design employs `normal` spacing (value `2`), providing adequate whitespace for visual comfort without feeling overly sparse or dense.
